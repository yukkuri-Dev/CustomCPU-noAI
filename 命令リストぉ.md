# 命令リスト

以下は、エミュレータでサポートされている命令の一覧です。

## 算術演算
- **ADD (0x00000001)**: `R4 = R1 + R2`
- **SUB (0x00000002)**: `R4 = R1 - R2`

## データ転送
- **MOV (0x00000010)**: `R[Operand1] = R[Operand2]`
- **LOAD (0x00000020)**: `R[Operand1] = RAM[Operand2]`
- **STORE (0x00000021)**: `RAM[Operand2] = R[Operand1]`

## ジャンプ命令
- **JMP (0x00000030)**: 条件に応じてPCを変更
  - モード0: `PC = R1`
  - モード1: `PC = R1 / 4`
- **JZ (0x00000031)**: 条件付きジャンプ
  - モード0: `if R2 == 0, PC = R1`
  - モード1: `if R2 == 0, PC = R1 / 4`

## 論理演算
- **AND (0x00000040)**: `R4 = R1 & R2`
- **OR (0x00000041)**: `R4 = R1 | R2`
- **XOR (0x00000042)**: `R4 = R1 ^ R2`
- **NOT (0x00000045)**: `R4 = ~R1`

## シフト演算
- **SHL (0x00000050)**: `R4 = R1 << R2`
- **SHR (0x00000051)**: `R4 = R1 >> R2`
- **ROL (0x00000052)**: `R4 = (R1 << R2) | (R1 >> (32 - R2))`
- **ROR (0x00000053)**: `R4 = (R1 >> R2) | (R1 << (32 - R2))`

## デバッグ用命令
- **DEBUG_PRINT (0x0FFFFFFF)**: 指定レジスタの値を出力
- **NOP (0x00000000)**: 何もしない
- **DEBUG_STOP (0xFFFFFFFF)**: エミュレータを停止

## 未定義命令
- 未定義の命令が実行されると、エラーが発生します。
